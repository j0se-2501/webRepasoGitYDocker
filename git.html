Ejercicios Git:

1. ¿Qué comando inicia un repositorio de Git?

git init

2. ¿Qué zonas lógicas se pueden identificar?

Working directory, staging area, repositorio local y repositorio remoto.

3. ¿Qué comando muestra los commits realizados?

git log

4. ¿Qué comandos crean una rama nueva?

git branch [nombre rama nueva], git checkout -b [nombre de la rama]

5. ¿Qué es una release de Github?

Iteraciones del software empaquetadas y disponibles para ser descargadas por los usuarios.

6. ¿Qué es un issue en Github?

Una nota en un repositorio que trata de llamar la atención sobre un problema.

7. ¿Cómo se llama el lenguaje de marcado que usa Github?

Markdown.

8. ¿Qué es una pull request?

Una solicitud de cambio del código, una petición para poder insertar modificaciones en una rama de un proyecto de git.
