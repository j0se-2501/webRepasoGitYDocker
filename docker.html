Ejercicios Docker:

1. ¿Qué es Docker?

Docker es una plataforma de software que permite la creación, el despliegue y la ejecución de aplicaciones en contenedores.

2. ¿Cuál es la diferencia entre una imagen y un contenedor en Docker?

Una imagen es una plantilla de solo lectura con instrucciones para crear un contenedor, mientras que un contenedor es una instancia en ejecución de una imagen.

3. ¿Cómo puedes exponer un puerto en un contenedor Docker?

Se puede exponer un puerto en un contenedor Docker utilizando la opción "-p" seguido del puerto del host y del puerto del contenedor.

4. ¿Cómo puedes detener y eliminar un contenedor en Docker?

Para detener un contenedor: `docker stop nombre_del_contenedor`. Para eliminar un contenedor: `docker rm nombre_del_contenedor`.

5. ¿Qué es un Dockerfile y para qué se utiliza?

Un Dockerfile es un archivo de texto que contiene instrucciones para construir una imagen Docker. Se utiliza para automatizar el proceso de creación de imágenes Docker.

6. ¿Cómo puedes ver las imágenes de Docker en tu sistema?

Se puede ver la lista de imágenes de Docker en el sistema utilizando el comando `docker images`.

7. ¿Qué es Docker Compose y para qué se utiliza?

Docker Compose es una herramienta que permite definir y gestionar aplicaciones Docker con múltiples contenedores. Se utiliza para simplificar el proceso de definir y ejecutar aplicaciones complejas.

8. ¿Cómo puedes construir una imagen Docker desde un Dockerfile?

Se puede construir una imagen Docker desde un Dockerfile utilizando el comando `docker build -t nombre_de_la_imagen ruta_del_Dockerfile`.
